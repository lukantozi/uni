# Credits

This project was inspired by [Jani Nikula](https://github.com/jnikula)'s [Hawkmoth](https://github.com/jnikula/hawkmoth) project, which provides Sphinx-based autodoc for C. The core idea of extracting C doc comments through libclang originates there.

mkdocs-cdoc adapts and extends that concept for the MkDocs ecosystem, adding multi-source-group navigation, cross-reference resolution, IGT test catalog generation, and gtk-doc migration tooling.
